Customer Churn: Perdict to save your business

Github Repository

I'm thrilled to introduce my latest project on Telco Customer Churn Prediction. This project focuses on predicting customer churn using machine learning techniques, a crucial aspect for businesses in understanding and retaining their customer base.

Project Overview

  • Data Cleaning and Preprocessing: Involved dropping unnecessary columns, converting strings to numerical values, and applying one-hot encoding for categorical data.
  • Data Visualization: Insights into customer tenure and monthly charges were visualized to understand customer behavior.
  • Model Building: A Sequential model with Dense layers was used, employing 'relu' and 'sigmoid' activations.

Model Performance

Conclusion

This project on Telco Customer Churn Prediction is a step towards understanding and leveraging machine learning to enhance customer retention strategies. The insights and methodologies used here can be a valuable resource for businesses and data enthusiasts alike.
