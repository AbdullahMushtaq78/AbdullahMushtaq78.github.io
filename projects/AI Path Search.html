Ai Cave: Ai Path finding algorithms

Github Repository

I'm excited to showcase my AI Searching Algorithms project, a comprehensive collection of various pathfinding and puzzle-solving algorithms implemented in Python. This project demonstrates the application of these algorithms in different scenarios, offering insights into their workings and efficiencies.

Featured Algorithms

  • A* Search Algorithm for 8-Puzzle: Implements the A* algorithm to solve 8-puzzle problems, considering both heuristic and actual costs. Read more
  • A* Algorithm Visualization: An interactive visualization of the A* algorithm using Pygame, demonstrating the exploration of optimal paths. Read more
  • Greedy Best First Search for 8-Puzzle: A demonstration of the Greedy Best First Search algorithm applied to 8-puzzle problems. Read more
  • Hill Climbing Algorithm for Sudoku: Solves Sudoku puzzles using the Hill Climbing algorithm, showcasing its heuristic-based approach. Read more
  • MiniMax Algorithm for Tic Tac Toe: An implementation of the MiniMax algorithm for a Tic Tac Toe AI agent. Read more
  • Uniform Cost Search for 8-Puzzle: Utilizes Uniform Cost Search to find the optimal solution in 8-puzzle problems. Read more
