Android Contacts Management Application in Java

Github Repository

I'm thrilled to introduce my first android application project, an Android Contacts App developed in Java with complete integration of SQLite database using Android Studio. This app is designed to manage contacts efficiently with a user-friendly interface and robust functionalities.

Key Features of the App

  • Comprehensive Contact Management: Users can add new contacts, edit existing ones, and delete contacts. There's also a feature to delete all contacts at once.
  • Search Functionality: The app includes a search feature that allows users to find contacts easily by entering their names.

User Interface and Experience

The app boasts a modern and aesthetic look with vibrant colors and a dark theme. A beautiful sliding effect is implemented to reveal the Edit and Delete buttons on each contact card.


Recent Enhancements

  • Phone Call Functionality: Added the ability to make phone calls directly from the app by clicking on the new Dialer Icon.
  • Text Messaging: Users can now send text messages directly through the app, enhancing communication convenience.
